## 게임 방식

Timmy Survival은 끝없이 밀려오는 적들을 피하고 처치하며 생존하는 탑다운 서바이벌 게임입니다.

- 스테이지별로 지정된 수의 적이 스폰됩니다.
- **모든 적을 처치하면** 다음 스테이지로 자동 진행됩니다.
- 스테이지 3 이후에는 **무한 모드**로 반복되며, 반복할수록 적 HP·스폰 수·스폰 속도가 증가합니다.
- 플레이어 HP가 0이 되면 **GAME OVER**입니다.

---

## 조작 방법

| 키 / 버튼 | 동작 |
|---|---|
| `W` / `A` / `S` / `D` | 이동 |
| `마우스 좌클릭 (LMB)` | Burst Strike 발동 (스킬 해금 후) |
| `마우스 우클릭 (RMB)` | Poison Gas Zone 배치 (스킬 해금 후) |
| `Q` | 상점(Shop) 열기 / 닫기 |
| `1` ~ `5` | 상점 아이템 구매 (상점 오픈 중) |
| `E` | 상점 아이템 리롤 (상점 오픈 중) |
| `R` | 화면 내 모든 코인 자석 수집 (쿨다운 60초) |
| `Tab` | 카메라 플레이어 ↔ 랜덤 적 전환 |
| `←` / `→` 방향키 | 게임 속도 조절 (0.1x ~ 5.0x) |
| `Space` | 게임 속도 1.0x 로 초기화 |
| `Enter` | 게임 오버 후 재시작 |

> 상점이 열려 있는 동안에는 게임이 **일시 정지**됩니다.

쿨다운이 충전되면 최대 **3개까지 스택**으로 보관할 수 있습니다.

| 지속 시간 | 0.45초 |
| 넉백 | 320 |

- Damage Up: 데미지 +25%

## 상점 시스템

- `Q`로 상점을 열고, 게임이 **일시 정지**됩니다.
- 매번 상점을 열 때 5개의 아이템이 무작위 제공됩니다.
- `1`~`5` 키로 아이템을 구매합니다.
- `E` 키로 아이템 목록을 **리롤**합니다 (첫 리롤 4코인, 이후 매번 +2코인).

### 스탯 업그레이드

| 아이템 | 비용 | 효과 |
|---|---|---|
| Move Speed Up | 8 | 이동 속도 +7% |
| Pickup Radius Up | 6 | 코인 흡수 반경 +15% |
| Small Heal | 8 | HP +25 회복 |
| Large Heal | 15 | HP +50 회복 |

## 코인 & 보상

- 일반 적 처치 시 코인 **1개** 드롭
- 보스 처치 시 코인 **18~42개** 폭발 드롭 (보스 HP 스케일에 비례)
- `R` 키: 화면 내 모든 코인을 즉시 자석처럼 흡수 (쿨다운 **60초**)
- 코인은 플레이어 흡수 반경(기본 50px) 내에 진입하면 자동 수집됩니다.
